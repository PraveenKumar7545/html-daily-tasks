# HTML Daily Task 03 - College Application Form

## 📌 Description

Created a College Application Form using HTML5.

This form collects basic student information such as personal details, contact information, gender, skills, department, date of birth, preferred time, favorite color, and file upload.

## 🛠️ Technologies Used

- HTML5

## 📚 HTML Concepts Covered

- HTML Forms
- `<form>`
- `<label>`
- Text Input
- Email Input
- Password Input
- Telephone Input
- Radio Buttons
- Checkboxes
- Select Dropdown
- Date Input
- Time Input
- Color Input
- File Upload
- Submit Button
- Reset Button
- `required` Attribute
- `for` and `id` Attributes

## ✨ Features

- Student personal information form
- Email and password fields
- Gender selection
- Multiple skills selection
- Department dropdown
- Date of birth selection
- Preferred time selection
- Favorite color selection
- File upload option
- Register and Reset buttons
