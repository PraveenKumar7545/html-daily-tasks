# HTML Daily Task 02 - Project Hub

## 📌 Description

Created a project hub webpage using HTML5. This webpage presents personal information, available academic projects, project descriptions, services, pricing details, and contact information.

## 🛠️ Technologies Used

* HTML5

## 📚 HTML Concepts Covered

* Basic HTML document structure
* Headings
* Paragraphs
* Text formatting tags
* Images
* Hyperlinks
* Ordered lists
* Description lists
* Audio element
* Video element
* Tables
* Horizontal lines
* `target` attribute
* `alt` attribute

## ✨ Features

* Personal profile section
* GitHub profile link
* Available projects list
* Project descriptions
* Services section
* Project pricing table
* Audio demo
* Video demo
* Contact information
