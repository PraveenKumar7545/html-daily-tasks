# HTML Daily Task 04 - SBI Fund Transfer Form

## 📌 Description

Created a simple SBI Fund Transfer webpage using HTML5.

This webpage allows users to select a bank account, enter a transfer amount, choose a payment purpose, select a beneficiary, choose a payment option, and submit or cancel the transfer form.

## 🛠️ Technologies Used

- HTML5

## 📚 HTML Concepts Covered

- HTML Forms
- Tables
- Radio Buttons
- Text Input
- Select Dropdown
- Options
- Submit Button
- Reset Button
- Labels
- Form Structure

## ✨ Features

- Account selection
- Account details displayed using a table
- Amount input field
- Payment purpose selection
- Beneficiary selection
- Beneficiary details table
- Payment option selection
- Submit and Cancel buttons

## 📊 Tables Used

### Account Details

- Account Number
- Account Type
- Branch
- Balance

### Beneficiary Details

- Account Number
- Beneficiary Name
- Branch
- Transfer Limit
